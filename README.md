# Breast Cancer Detection 🎗️
This project predicts whether a tumor is malignant or benign using machine learning techniques.
# Features:
- Data preprocessing and cleaning
- Feature selection
- Classification model implementation
- Model evaluation and accuracy improvement
# Technologies Used:
- Python
- Pandas
- NumPy
- Scikit-learn
# How to Run:
- Open the .ipynb file in Google Colab or Jupyter Notebook
- Run all the cells
# Project Objective:
To develop a predictive model that helps in early detection of breast cancer using medical data.
